# [Joshua] – CAD & BIM Portfolio

A modern, responsive portfolio website.

Live site: [https://yourfriend-portfolio.com](https://yourfriend-portfolio.com) 


## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling single-page layout
- Professional sections: About, Skills, Projects (or Experience), Contact
- Functional contact form with email delivery (powered by Web3Forms)
- Clean, minimal aesthetic focused on content and readability

## Technologies Used

- **React** + **Vite** – Fast modern frontend framework
- **JavaScript (ES6+)**
- **CSS Modules** – Scoped and maintainable styling
- **Web3Forms** – Serverless contact form backend
- Hosted on [Render]
